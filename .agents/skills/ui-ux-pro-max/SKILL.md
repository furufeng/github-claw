# UI-UX-PRO-MAX 技能

## 技能名称
ui-ux-pro-max

## 用途
为项目生成高颜值、移动优先的导航官网/落地页，采用现代 UI 设计规范：
- Glassmorphism 玻璃质感卡片
- 渐变配色与动态粒子背景
- 流畅 CSS 动画与过渡
- 响应式布局（移动端优先）
- 无障碍（WCAG AA 对比度合规）
- 纯 HTML/CSS/JS，零依赖，可直接部署到 GitHub Pages

## 输入
- 产品名称与 Slogan
- 核心特性列表
- 用户场景/对比信息
- 品牌色偏好（可选）

## 输出
- `docs/index.html`：完整单文件落地页
- 可直接上传 GitHub Pages 或任意静态托管服务

## 依赖
无外部依赖，仅需现代浏览器（Chrome 90+、Safari 14+）
